# UML Diagrams для S3 Storage Application

Этот каталог содержит UML диаграммы проекта в формате PlantUML.

## Файлы диаграмм

1. **class-diagram.puml** - Диаграмма классов
   - Показывает структуру классов и их взаимосвязи
   - Frontend и Backend компоненты

2. **sequence-upload.puml** - Последовательность операций при загрузке файла
   - Взаимодействие между компонентами при загрузке

3. **sequence-list.puml** - Последовательность операций при получении списка файлов
   - Как работает получение списка файлов

4. **sequence-download.puml** - Последовательность операций при скачивании файла
   - Процесс скачивания файла

5. **component-diagram.puml** - Диаграмма компонентов
   - Архитектура компонентов системы

6. **activity-diagram.puml** - Диаграмма активности
   - Процесс загрузки файла

7. **state-diagram.puml** - Диаграмма состояний
   - Состояния приложения и переходы

8. **deployment-diagram.puml** - Диаграмма развертывания
   - Размещение компонентов в системе

9. **usecase-diagram.puml** - Диаграмма прецедентов использования
   - Пользовательские сценарии

## Описание диаграмм

### Диаграмма классов
Показывает структуру классов приложения, их атрибуты, методы и отношения между классами.

### Диаграммы последовательности
Показывают последовательность взаимодействий между объектами во времени для конкретных сценариев использования.

### Диаграмма компонентов
Отображает архитектуру системы на уровне компонентов и их взаимосвязей.

### Диаграмма активности
Моделирует бизнес-процессы и потоки работ в системе.

### Диаграмма состояний
Показывает различные состояния системы и переходы между ними.

### Диаграмма развертывания
Отображает физическое развертывание компонентов системы.

### Диаграмма прецедентов
Показывает функциональные требования системы с точки зрения пользователя.

## Генерация изображений

### Автоматическая генерация (рекомендуется)

Используйте скрипт `generate-png.sh` для автоматической генерации всех PNG:
```bash
cd uml
./generate-png.sh
```

Скрипт использует онлайн API PlantUML и не требует установки дополнительного ПО.

### Ручная генерация

Если хотите генерировать локально:

```bash
# Установка PlantUML (требуется Java)
# Ubuntu/Debian:
sudo apt-get install plantuml

# Или через npm:
npm install -g node-plantuml

# Генерация всех диаграмм:
plantuml uml/*.puml

# Генерация конкретной диаграммы:
plantuml uml/class-diagram.puml
```

## Формат файлов

Все диаграммы используют формат PlantUML - текстовый формат для создания UML диаграмм. Это позволяет:
- Хранить диаграммы в системе контроля версий (Git)
- Легко редактировать и поддерживать
- Генерировать изображения в различных форматах
- Интегрироваться с различными инструментами
